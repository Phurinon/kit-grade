# 📊 KIT-GRADE (เครื่องมือคิดเกรด) https://phurinon.github.io/kit-grade/

โปรแกรมเว็บแอปพลิเคชันสำหรับคำนวณและจัดการเกรด ออกแบบมาให้ใช้งานง่าย รองรับทั้งระบบ **คุณครู (ผู้จัดการห้องเรียน)** และ **นักเรียน (ผู้ติดตามผลการเรียนส่วนตัว)** พัฒนาด้วยเทคโนโลยีที่ทันสมัยล่าสุด (React 19 + Vite + TailwindCSS v4)

## ✨ คุณสมบัติหลัก (Features)

### 👨‍🏫 โหมดคุณครู (Teacher Mode)
- **การจัดการรายวิชา:** สร้างรายวิชา กำหนดรูปแบบการตัดเกรด (เปอร์เซ็นต์ / เกรด A-F)
- **การจัดการชิ้นงาน:** เพิ่ม ลบ หรือแก้ไขชื่อและ **คะแนนเต็ม** ของชิ้นงานได้อย่างอิสระ
- **นำเข้ารายชื่อนักเรียน (Excel Import):** 
  - สร้างวิชาพร้อมอัปโหลดไฟล์ Excel เพื่อนำเข้ารายชื่อนักเรียนได้ทันที
  - ระบบอ่านคอลัมน์ A อัตโนมัติ (และจะดึงชื่อจากคอลัมน์ B ให้หากคอลัมน์ A เป็นเพียงเลขที่)
- **ตารางคะแนนอัจฉริยะ (Score Matrix):** กรอกคะแนนดิบของนักเรียนทุกคนได้ในหน้าต่างเดียว พร้อมคำนวณเกรด/เปอร์เซ็นต์ให้แบบ Real-time
- **ส่งออกรายงาน (Excel Export):** แตกไฟล์ตารางคะแนนทั้งหมดออกเป็น `.xlsx` เพื่อนำไปปริ้นท์หรือส่งต่อได้อย่างสวยงามและรองรับภาษาไทย 100%

### 👨‍🎓 โหมดนักเรียน (Student Mode)
- **ติดตามเกรดส่วนตัว:** บันทึกและคำนวณเกรดของแต่ละวิชาได้อย่างแม่นยำ
- **คำนวณเกรดเฉลี่ยสะสม (GPA):** ระบบจะนำเกรดของทุกวิชามาคำนวณเป็น GPA ให้อัตโนมัติ (อิงตามมาตรฐาน 4.0)
- **ระบบคะแนนมาตรฐาน:** ใช้ระบบ (ผลรวมคะแนนที่ได้ ÷ ผลรวมคะแนนเต็ม) ทำให้คำนวณตรงกับโหมดคุณครูเป๊ะๆ

### 💾 ระบบจัดการข้อมูล (Data Management)
- **ทำงานบน Browser 100%:** รวดเร็ว ปลอดภัย ข้อมูลไม่รั่วไหล (ไม่ต้องพึ่งพาเซิร์ฟเวอร์)
- **Backup / Restore:** สำรองข้อมูลทั้งแอปพลิเคชันเก็บไว้เป็นไฟล์ `.json` ในเครื่อง และกู้คืนได้ทุกเมื่อที่ต้องการ

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)
- **Frontend Framework:** React 19
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4 (ระบบ Glassmorphism UI)
- **Build Tool:** Vite
- **Data Processing:** `xlsx` (สำหรับการจัดการไฟล์ Excel)

## 🚀 การติดตั้งและใช้งาน (Installation)

### 1. รันบนเครื่องส่วนตัว (Local Development)
```bash
# ติดตั้ง Packages
npm install

# รันเซิร์ฟเวอร์จำลอง
npm run dev
```
จากนั้นเปิด Browser ไปที่ `http://localhost:5173`

### 2. การนำขึ้นระบบออนไลน์ (Deploy to GitHub Pages)
โปรเจกต์นี้ถูกตั้งค่ามาให้พร้อมนำขึ้น GitHub Pages ฟรีได้ทันที
1. อัปโหลดโค้ดขึ้น Repository บน GitHub
2. รันคำสั่งด้านล่างนี้ใน Terminal:
```bash
npm run deploy
```
ระบบจะทำการ Build โค้ดและอัปโหลดไปไว้บน GitHub Pages ของคุณให้โดยอัตโนมัติ!

## 📁 โครงสร้างโปรเจกต์ (Project Structure)
```text
src/
 ├── types.ts                     # แหล่งรวม TypeScript Interfaces ทั้งหมด (Data Models)
 ├── App.tsx                      # ตัวจัดการ Global State (AppData), การ Backup/Restore และ Routing
 ├── index.css                    # ตั้งค่า Tailwind CSS และ Print Styles
 └── components/                  # โฟลเดอร์เก็บชิ้นส่วนหน้าต่าง UI
      ├── WelcomeScreen.tsx       # หน้าแรกสำหรับเลือกโหมด (Student/Teacher)
      ├── TeacherApp.tsx          # ระบบของโหมดคุณครู (ตารางคะแนน, การ Import/Export Excel)
      ├── StudentApp.tsx          # ระบบของโหมดนักเรียน (กระดานสรุป GPA)
      └── StudentSubjectCard.tsx  # การ์ดรายวิชาสำหรับโหมดนักเรียน
```

## 📝 หมายเหตุการใช้งาน
- การคำนวณเปอร์เซ็นต์อ้างอิงจากสูตร: `(ผลรวมคะแนนที่ได้ ÷ ผลรวมคะแนนเต็ม) × 100`
- การนำเข้า/ส่งออกด้วยไฟล์ Excel สนับสนุนไฟล์ `.xlsx` และ `.xls`
- ทุกข้อมูลจะถูกเคลียร์หากคุณเปลี่ยนเบราว์เซอร์หรือลบข้อมูลใน Local (ควรใช้ปุ่ม "สำรองข้อมูลระบบ" เป็นประจำ)
