# 📊 เครื่องคิดเกรดนักเรียนมัธยม (Grade Calculator)

โปรแกรมเว็บแอพสำหรับคำนวณเกรดของนักเรียนมัธยม พัฒนาด้วย React + TypeScript + Vite

## ✨ คุณสมบัติ

### 📚 การจัดการวิชา (Subject Management)
- ✅ เพิ่มและลบวิชาได้ไม่จำกัด
- ✅ เลือกประเภทการคะแนนสำหรับแต่ละวิชา:
  - **เปอร์เซ็นต์ (0-100)**: สำหรับคะแนนตัวเลข
  - **เกรด A-F**: สำหรับระบบเกรดตัวอักษร
  - **ยืดหยุ่น**: แต่ละวิชาสามารถเลือกประเภทได้อย่างอิสระ

### 📝 การจัดการงาน (Assignment Management)
- ✅ เพิ่มจำนวนงานไม่จำกัดต่อวิชา
- ✅ กำหนด **ค่าน้ำหนัก (Weight)** สำหรับแต่ละงาน
  - ตัวอย่าง: สอบกลางภาค 40%, สอบปลายภาค 50%, งานที่บ้าน 10%
- ✅ แก้ไขและลบงานได้
- ✅ คะแนนช่วง 0-100 สำหรับทุกประเภท

### 📊 การคำนวณเกรด (Grade Calculation)
- ✅ **คำนวณเกรดถ่วงน้ำหนัก (Weighted Average)**:
  - `เกรด = (Σ คะแนน × น้ำหนัก) ÷ Σ น้ำหนัก`
- ✅ **คำนวณ GPA อัตโนมัติ** (ตามมาตรฐาน 4.0 GPA):
  - A (90-100) = 4.0
  - B (80-89) = 3.0
  - C (70-79) = 2.0
  - D (60-69) = 1.0
  - F (< 60) = 0.0
- ✅ แสดง GPA รวมของนักเรียน

### 💾 การบันทึกและนำเข้าข้อมูล (Save/Import)
- ✅ **Export JSON**: ดาวน์โหลดข้อมูลเป็นไฟล์ JSON
- ✅ **Import JSON**: นำเข้าข้อมูลจากไฟล์ JSON ที่บันทึกไว้
- ✅ ข้อมูลมีวันที่ในชื่อไฟล์

### 🎨 ส่วนตัวเลขและ UI/UX
- ✅ ดีไซน์สมัยใหม่และสวยงาม
- ✅ Responsive Design - ใช้ได้บนมือถือ แท็บเล็ต และเดสก์ทอป
- ✅ Animation ที่ราบรื่น
- ✅ ดาร์กมโด (Dark Mode) รองรับโดยอัตโนมัติ
- ✅ ข้อผิดพลาดที่ชัดเจน

## 🚀 วิธีใช้

### ติดตั้ง
```bash
npm install
```

### รันแอพในโหมดพัฒนา (Development)
```bash
npm run dev
```
เปิด `http://localhost:5173` ในเบราวเซอร์

### Build สำหรับ Production
```bash
npm run build
```

### Preview Build
```bash
npm run preview
```

## 📖 ขั้นตอนการใช้งาน

1. **เพิ่มวิชา**: 
   - ป้อนชื่อวิชา (เช่น คณิตศาสตร์, วิทยาศาสตร์)
   - เลือกประเภทคะแนน (ตัวเลขหรือเกรด)
   - คลิก "เพิ่มวิชา"

2. **เพิ่มงาน**:
   - ในการ์ดวิชา คลิก "เพิ่มงาน"
   - ป้อนชื่องาน (เช่น โครงงาน, ทดสอบย่อย)
   - ป้อนคะแนน (0-100)
   - กำหนดค่าน้ำหนัก (default: 1)
   - คลิก "เพิ่มงาน"

3. **ดูเกรด**:
   - เกรดอัตโนมัติแสดงบนการ์ดวิชา
   - GPA รวมแสดงด้านบนหน้า

4. **บันทึกข้อมูล**:
   - คลิก "Export" เพื่อดาวน์โหลดข้อมูล
   - คลิก "Import" เพื่อโหลดข้อมูลที่บันทึกไว้

## 🛠️ เทคโนโลยี

- **React 19**: JavaScript library สำหรับสร้าง UI
- **TypeScript**: ภาษา JavaScript ที่มีความปลอดภัยตรวจสอบประเภท
- **Vite**: Build tool ที่รวดเร็ว
- **CSS3**: สำหรับการจัดแต่ง
